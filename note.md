##### vue项目搭建步骤
```html
$ vue init webpack my-project
$ cd my-project
$ npm install
$ npm run dev
```
##### vue-cli 3.0 搭建Vue项目

```
1. 安装vue-cli 3.0
   npm i @vue/cli -g
2. vue-cli搭建脚本文件
   vue create vue-test
```
